cDashboard
✉

cDashboard is an open source information based overlay for Microsoft Windows, written in C#. It allows the user to access various 'cForms' on an interface completely separate from the normal Windows UI, accessible via a simple keyboard shortcut.

Give cDash a try, download and run the exe. If you really like it, move it to your 'Startup' folder! x64/x86 .exe
Check out the full source code on GitHub.

cSticky

cSticky was the cForm that started it all. I had wanted a way to access sticky notes in a way similar to the way the Steam overlay works. It supports full rich text formatting among other customization options. The user can select any color, size for the cSticky and any color, font, size, formatting for text. In addition, the user has the option to export a cSticky's text to an .rtf file. Keyboard Shortcuts: Ctrl-T: New cSticky of the user's favorite color, Ctrl-S: Strikethrough selected text, Ctrl-B: Bold selected text, Ctrl-U: Underline selected text, Ctrl-I: Italicize selected text

cPic

cPic is a slideshow viewer cForm. It allows the user to select an unlimited amount of images (png, jpeg, gif) that will be cycled through every time the dash fades in. In addition, it allows the user to select from various layouts to see their images just the way they like them. It is plain, by design, so that the images are front and center.

cStopwatch

cStopwatch is a stopwatch cForm. It will count up from zero as time goes by. The user may 'lap' and pause the stopwatch as often as they like. If cDashboard is not running, or the computer is turned off, the stopwatch will still give an accurate representation of time gone by. S to start and stop, L to lap, R to reset.

cWeather

cWeather is a weather information cForm. It can retrive accurate, up-to-date weather information via the Yahoo Weather API, automatically, every hour, or whenever you click "Refresh Data." It gives the current temperature, humidity, conditions, wind, a prediction for high, low, later conditions. You can have cWeathers for as many locations as you like.

cBattery

cBattery is a battery information cForm. It displays (via text and a graphical representation) remaining battery life percentage. If available, it also displays an approximate amount of battery time left on the current charge.

cMote

cMote is a remote control cForm. It can control various multimedia applications by playing, pausing, and changing the currently playing track. In addition, the system volume can also be controlled. The checkbox in the upper-left corner corresponds to Spotify integration, which can try to get and display album artwork and song information, from the internet, for the currently playing song in Spotify. Spotify integration requires internet connectivity.

cReminder

cReminder is a reminder setup cForm. It allows the creation of cReminders with specific date, time, and message. After set, a global cNotification will be shown to the user with the saved message upon elapse.

cRViewer

cRViewer is a management cForm for cReminders. The user can view all pending cReminders' times, dates, and messages. In addition any cReminder can be removed (deleted) if no longer needed. It also provides an added means of creating new cReminders.

cNotification

cNotification is an extendable global notification cForm. It is used to display cReminders and their messages when they come due, via a smooth fade in (and fade out.) In addition, display time is user-customizable.
